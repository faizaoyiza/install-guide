# Tech Environment Setup

## Introduction

This project provides a comprehensive guide to setting up a complete **DevOps learning environment**. It is designed to equip students enrolled in the DevOps program with all the necessary tools and accounts required for hands-on labs and projects.

The guide covers the installation of essential software tools such as **Git**, **Visual Studio Code (VS Code)**, **Oracle VirtualBox**, and **Ubuntu Linux**, as well as the creation of required accounts like **GitHub** and **Amazon Web Services (AWS)**. All downloads and account setups are linked to official sources to ensure security and authenticity.

---

## Project Prerequisites

Before starting this setup, ensure the following requirements are met:

- Completion of the **Tech Environment Setup** introductory courses  
- A stable **internet connection** for downloading software and accessing cloud services  
- A computer system with:
  - Minimum **8GB RAM** (recommended)
  - **64-bit architecture** (highly recommended)
  - Sufficient disk space to run virtual machines and containers

## Installing Software Tools

Since the Tech Environment Setup videos have been completed, the next step is to install and configure the required tools on your computer.

---

### 1. Install Visual Studio Code (VS Code)

Visual Studio Code is a lightweight and powerful code editor used for scripting, configuration, and development tasks.

**Steps:**
1. Visit the official website:  
   https://code.visualstudio.com/
2. Download the installer for your operating system.
3. Run the installer and follow the installation prompts.
4. Launch VS Code after installation to confirm it works correctly.

## 3. Install Oracle VirtualBox

Oracle VirtualBox is used to run virtual machines on your local system.

### Steps:
- Download VirtualBox from: [https://www.virtualbox.org/](https://www.virtualbox.org/)
- Install the version suitable for your operating system.
- Download the VirtualBox Extension Pack from the same website.
- Install the Extension Pack to enable additional features.

## 4. Install Ubuntu on VirtualBox (Windows & macOS â€“ Intel)

Ubuntu Linux will serve as the primary development and server environment.

### Steps:
- Download the Ubuntu Desktop ISO file from: [https://ubuntu.com/download/desktop](https://ubuntu.com/download/desktop)
- Open VirtualBox and click **New**.
- Configure the virtual machine:
  - **Name:** Ubuntu  
  - **Type:** Linux  
  - **Version:** Ubuntu (64-bit)
- Allocate system resources:
  - **Memory:** At least 4096 MB (4GB)  
  - **CPU:** 2 cores or more
- Attach the Ubuntu ISO file and start the virtual machine.
- Follow the on-screen instructions to complete the Ubuntu installation.

## Account Setup

### 1. Create a GitHub Account

GitHub is used for source code management, collaboration, and hosting repositories.

#### Steps:
- Visit: [https://github.com/](https://github.com/)
- Sign up using your email address.
- Verify your email.
- Create a new repository for your DevOps projects.

### 2. Create an AWS Account

Amazon Web Services (AWS) provides cloud infrastructure and services used throughout the DevOps program.

#### Steps:
- Visit: [https://aws.amazon.com/](https://aws.amazon.com/)
- Click **Create an AWS Account**.
- Provide required personal and billing details (AWS Free Tier eligible).
- Verify your identity.
- Enable **Multi-Factor Authentication (MFA)** for security.
